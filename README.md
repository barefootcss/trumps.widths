#Widths Module

The `widths` module is a helper class for setting widths on elements.

	<div class="grid">
		<div class="grid__item w-1-3 sw-1-1"></div>
		<div class="grid__item w-1-3 sw-1-1"></div>
		<div class="grid__item w-1-3 sw-1-1"></div>
	</div>

## Installation

Install using Bower:

	$ bower --save bare-widths

Import into project:

	@import "bower_components/bare-widths/trumps.widths";
